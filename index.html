What this does

This services is designed to help you explore PeopleSoft App Designer files. Simply drag the XML generated from the 'Copy Project > To File...' menu option in App Designer and the results list will populate with the list of items in the file. Click on the name of an element to view the contents of that object. This project and site is owned, operated, and maintained by Kyle Shaver.

Requires a modern browser, like Google Chrome or Mozilla Firefox.
